




<h1 class="heading"><span class="name">Catenate First</span><span class="command">R←X⍪[K]Y</span></h1>



The form `R←X⍪Y` implies catenation along the first axis whereas the form `R←X,Y` implies catenation along the last axis (columns).  See [Catenate/Laminate: above](catenate-laminate.md).
