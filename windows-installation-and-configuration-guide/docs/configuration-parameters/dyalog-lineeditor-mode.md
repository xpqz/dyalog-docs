<h1 class="heading"><span class="name">Dyalog_LineEditor_Mode</span></h1>

This Boolean parameter specifies whether or not multi-line input is enabled in the Session.

See also [Multiline Input](../configuring-the-ide/configuration-dialog/configuration-dialog-session-tab.md) and [Multi-line Session Input](../../../windows-ui-guide/multiline-session-input).
