<h1 class="heading"><span class="name">Trace_First_Y</span></h1>

This parameters specifies the initial vertical position on the screen of the *first* trace window in character units. Subsequent trace windows will be staggered. This applies only if **ClassicMode** is 1.

See also [Y Pos](../configuring-the-ide/configuration-dialog/configuration-dialog-windows-tab.md).
