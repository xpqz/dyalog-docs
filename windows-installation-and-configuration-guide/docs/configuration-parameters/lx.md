<h1 class="heading"><span class="name">LX</span></h1>

This parameter specifies an expression to be executed after Dyalog has started and loaded a workspace or a text file containing APL source code. Also see [ Load](load.md). This expression is run only on Dyalog start-up and overrides the workspace latent expression `âŽ•LX`.

The LX parameter applies only to the development version of Dyalog and is ignored in run-time applications.

The LX parameter is ignored when a workspace is loaded other than at start-up of the Dyalog program.

The LX parameter applies only to the Unicode edition of Dyalog and is ignored in Classic edition.
