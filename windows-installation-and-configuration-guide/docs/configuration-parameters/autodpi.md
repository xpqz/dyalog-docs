<h1 class="heading"><span class="name">AutoDPI</span></h1>

This parameter determines whether or not the Dyalog program registers the application as DPI-Aware when it initialises. If 1, (the default), Dyalog performs the auto-scaling; if 0, scaling is the responsibility of the programmer or operating system. See also [Enable DPI Scaling of the interpreter and development environment](../configuring-the-ide/configuration-dialog/configuration-dialog-general-tab.md).
