<h1 class="heading"><span class="name">InitialKeyboardLayoutShowAll</span></h1>

**Unicode Edition only.**

This Boolean parameter specifies whether or not all installed keyboards are listed in the choice of keyboards in the Configuration dialog box (Unicode Input tab).

See also [Show keyboards for all Languages](../configuring-the-ide/configuration-dialog/configuration-dialog-unicode-input-tab-unicode-edition-only.md).
