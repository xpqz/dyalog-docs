**Description**


The PageActiveObject property specifies a ref to the current [PropertyPage](../objects/propertypage.md) in a [PropertySheet](../objects/propertysheet.md). You may select a particular page by setting this property or by generating a [PageActivate](../methodorevents/pageactivate.md) event.
