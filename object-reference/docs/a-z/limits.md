|-----------|----------------------------------------|------------------------------|--------------------------------|----------------------------------|------------------------------|

**Description**


This property is a 2-element vector that specifies the minimum and maximum values of an object. The values must be in the range `(-2*31)` to `(¯1+2*31)` inclusive.
