<h1 class="heading"><span class="name">Port Number</span></h1>



A port number identifies the service provided by a particular server. It is an integer number in the range 0-65536,


Port numbers in the range 0-1023 are reserved for "well-known" services (for example, FTP and HTTP).


Port number 1024 is reserved for IANA.


Port numbers 1025-5000 are typically assigned for user-defined services.
