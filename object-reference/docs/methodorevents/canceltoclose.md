<h1 class="heading"><span class="name">CancelToClose</span> <span class="right">Method 367</span></h1>



**Applies To:** [PropertySheet](../objects/propertysheet.md)

**Description**


This method is used to change the buttons in a [PropertySheet](../objects/propertysheet.md) object. Its effect is to disable the Cancel button and, if the Style of the [PropertySheet](../objects/propertysheet.md) is `'Standard'`, it changes the text of the OK button to "Close". There is no result.


The CancelToClose method is niladic.
