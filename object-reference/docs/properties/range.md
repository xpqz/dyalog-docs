<h1 class="heading"><span class="name">Range</span> <span class="right">Property</span></h1>



**Applies To:** [Form](../objects/form.md), [Scroll](../objects/scroll.md), [SubForm](../objects/subform.md)

**Description**


This property determines the maximum value of the thumb in a scrollbar (the minimum value is always 1). This may be any positive integer value that is greater than 1.


For a [Scroll](../objects/scroll.md) object Range is a single number. For a [Form](../objects/form.md) or [SubForm](../objects/subform.md) object, Range is a 2-element vector which specifies the maxima for the [Form](../objects/form.md)'s vertical and horizontal scrollbars respectively.
