<h1 class="heading"><span class="name">ImageIndex</span> <span class="right">Property</span></h1>

[**Applies To**](../propertyapplies/imageindex.md)

**Description**


For a [ComboEx](../objects/comboex.md), [ListView](../objects/listview.md) or [TreeView](../objects/treeview.md), the ImageIndex property maps bitmapped images in an [ImageList](../objects/imagelist.md) to items. ImageIndex is an integer vector whose length is the same as the number of items in the object. See also [SelImageIndex](selimageindex.md)


For a [CoolBand](../objects/coolband.md), [MenuItem](../objects/menuitem.md), [TabButton](../objects/tabbutton.md) or [ToolButton](../objects/toolbutton.md), ImageIndex specifies the picture to be displayed in the object. In these cases, ImageIndex is a single integer value.
