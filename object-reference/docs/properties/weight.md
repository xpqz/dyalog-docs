<h1 class="heading"><span class="name">Weight</span> <span class="right">Property</span></h1>



**Applies To:** [Font](../objects/font.md)

**Description**


This property specifies the degree of boldness of a font associated with a [Font](../objects/font.md) object. It is a number in the range 0 to 1000, where 0 represents *very faint* and 1000 represents *very bold*. There is no default; the value of this property reflects the degree of boldness of the font allocated by Windows.


In general, 400 means *normal* and 700 means *bold*.
