<h1 class="heading"><span class="name">DateTime</span> <span class="right">Property</span></h1>



**Applies To:** [DateTimePicker](../objects/datetimepicker.md)

**Description**


Specifies the value of date/time in a [DateTimePicker](../objects/datetimepicker.md).


The DateTime property represents the date and time value that is currently displayed in a [DateTimePicker](../objects/datetimepicker.md) object.


It is normally a 4-element integer vector containing the date (as an IDN), hour, minutes and seconds respectively.
