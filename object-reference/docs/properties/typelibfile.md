<h1 class="heading"><span class="name">TypeLibFile</span> <span class="right">Property</span></h1>



**Applies To:** [ActiveXControl](../objects/activexcontrol.md), [OLEServer](../objects/oleserver.md)

**Description**


The TypeLibFile property is a read-only property that reports the name of the file in which the Type Library associated with a COM object is stored.
