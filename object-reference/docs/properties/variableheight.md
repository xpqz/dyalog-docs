<h1 class="heading"><span class="name">VariableHeight</span> <span class="right">Property</span></h1>



**Applies To:** [CoolBar](../objects/coolbar.md)

**Description**


The VariableHeight property specifies whether or not a [CoolBar](../objects/coolbar.md) displays bands at the minimum required height, or all the same height (that of the largest).


VariableHeight is a single number with the value 0 (same height) or 1 (variable height). The default is 1.
