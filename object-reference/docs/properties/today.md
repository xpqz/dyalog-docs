<h1 class="heading"><span class="name">Today</span> <span class="right">Property</span></h1>



**Applies To:** [Calendar](../objects/calendar.md), [DateTimePicker](../objects/datetimepicker.md)

**Description**


The Today property is an [IDN](../miscellaneous/international-day-number.md) that specifies today's date in a [Calendar](../objects/calendar.md) or [DateTimePicker](../objects/datetimepicker.md) object. Its default value is the current date that is set on your computer.


See also [CircleToday](circletoday.md) and [HasToday](hastoday.md) properties.
