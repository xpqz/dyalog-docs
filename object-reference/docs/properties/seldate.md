<h1 class="heading"><span class="name">SelDate</span> <span class="right">Property</span></h1>



**Applies To:** [Calendar](../objects/calendar.md)

**Description**


The SelDate property identifies the range of dates that is currently selected in a [Calendar](../objects/calendar.md) object.


SelDate is a 2-element integer vector of [IDN](../miscellaneous/international-day-number.md) values that identifies the first and last dates that are currently selected.
