<h1 class="heading"><span class="name">HasHelp</span> <span class="right">Property</span></h1>



**Applies To:** [PropertyPage](../objects/propertypage.md), [PropertySheet](../objects/propertysheet.md)

**Description**


The HasHelp property is a Boolean value   For a [PropertySheet](../objects/propertysheet.md), it determines whether or not the [PropertySheet](../objects/propertysheet.md) has a Help button. for a [PropertyPage](../objects/propertypage.md), HasHelp determines whether or not the Help button is active when the [PropertyPage](../objects/propertypage.md) is the current page. If the HasHelp property of a [PropertyPage](../objects/propertypage.md) is 0, the Help button on the parent [PropertySheet](../objects/propertysheet.md) will be temporarily disabled when that [PropertyPage](../objects/propertypage.md) is displayed.
