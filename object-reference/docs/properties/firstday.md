<h1 class="heading"><span class="name">FirstDay</span> <span class="right">Property</span></h1>



**Applies To:** [Calendar](../objects/calendar.md)

**Description**


The FirstDay property specifies the day that is considered to be the first day of the week and which appears first in the [Calendar](../objects/calendar.md).


FirstDay is an integer whose value is in the range 0-6. The default value for FirstDay depends upon your International Settings, but in most countries is 0 meaning Monday.
