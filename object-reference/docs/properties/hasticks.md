<h1 class="heading"><span class="name">HasTicks</span> <span class="right">Property</span></h1>



**Applies To:** [TrackBar](../objects/trackbar.md)

**Description**


The HasTicks property specifies whether or not tick marks are drawn in a [TrackBar](../objects/trackbar.md) object. It is Boolean value with a default value of 0.


The position of the tick marks in the [TrackBar](../objects/trackbar.md) is determined by the [TickAlign](tickalign.md) property.
