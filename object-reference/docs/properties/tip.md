<h1 class="heading"><span class="name">Tip</span> <span class="right">Property</span></h1>

[**Applies To**](../propertyapplies/tip.md)

**Description**


The Tip property is a character vector or character matrix that specifies a "help" message which is to be displayed when the user positions the mouse pointer over the object. The Tip is displayed in a pop-up [TipField](../objects/tipfield.md) object specified by the [TipObj](tipobj.md) property.
