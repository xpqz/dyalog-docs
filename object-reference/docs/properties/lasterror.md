<h1 class="heading"><span class="name">LastError</span> <span class="right">Property</span></h1>



**Applies To:** [ActiveXControl](../objects/activexcontrol.md), [OLEClient](../objects/oleclient.md), [OLEServer](../objects/oleserver.md), [Root](../objects/root.md)

**Description**


The LastError property provides information about the most recent error reported by OLE. You may use this property to report an error from an [OLEServer](../objects/oleserver.md) or [ActiveXControl](../objects/activexcontrol.md) to a host application.
