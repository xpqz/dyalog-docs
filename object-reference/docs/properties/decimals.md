<h1 class="heading"><span class="name">Decimals</span> <span class="right">Property</span></h1>



**Applies To:** [ButtonEdit](../objects/buttonedit.md), [Edit](../objects/edit.md), [Label](../objects/label.md), [Spinner](../objects/spinner.md)

**Description**


This property specifies the number of decimal places to which a number is to be displayed in an [Edit](../objects/edit.md) or [Label](../objects/label.md) object with [FieldType](fieldtype.md) `'Numeric'`. For an [Edit](../objects/edit.md) object, Decimals also specifies the maximum number of digits that the user may enter after a decimal point.


The default value of decimals is `Â¯1` which allows any number of decimal places to be entered.
