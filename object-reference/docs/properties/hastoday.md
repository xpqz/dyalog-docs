<h1 class="heading"><span class="name">HasToday</span> <span class="right">Property</span></h1>



**Applies To:** [Calendar](../objects/calendar.md), [DateTimePicker](../objects/datetimepicker.md)

**Description**


The HasToday property specifies whether or not the [Today](today.md) date is displayed in the bottom left corner of a [Calendar](../objects/calendar.md) object or in the drop-down calendar of a [DateTimePicker](../objects/datetimepicker.md).


HasToday is a single number with the value 0 (the date is *not* shown) or 1 (the date *is* shown); the default is 1.


See also [CircleToday](circletoday.md) property.
