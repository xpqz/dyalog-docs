<h1 class="heading"><span class="name">CurCell</span> <span class="right">Property</span></h1>



**Applies To:** [Grid](../objects/grid.md)

**Description**


This property specifies or reports the current cell in a [Grid](../objects/grid.md) object. The current cell is the one that is currently addressed by the user. The current cell is identified visibly if it has an associated object, other than an external.[ Label](../objects/label.md) object.
