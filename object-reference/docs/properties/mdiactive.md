<h1 class="heading"><span class="name">MDIActive</span> <span class="right">Property</span></h1>



**Applies To:** [MDIClient](../objects/mdiclient.md)

**Description**


This property contains the name of the [SubForm](../objects/subform.md) owned by the [MDIClient](../objects/mdiclient.md) that is currently active. Only one [SubForm](../objects/subform.md) may be active at a time. You can switch between [SubForm](../objects/subform.md)s in an MDI application under program control by setting this property with [`âŽ•WS`](../../../language-reference-guide/system-functions/ws). You can also do this by generating an [MDIActivate](../methodorevents/mdiactivate.md) event.


See also [MDIActiveObject](mdiactiveobject.md) property.
