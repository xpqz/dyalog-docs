<h1 class="heading"><span class="name">Italic</span> <span class="right">Property</span></h1>



**Applies To:** [Font](../objects/font.md)

**Description**


This property specifies whether or not a font represented by a [Font](../objects/font.md) object is italicised or not. It is either 0 (normal) or 1 (italic). There is no default; the value of this property reflects the characteristic of the selected font.
