<h1 class="heading"><span class="name">MinDate</span> <span class="right">Property</span></h1>



**Applies To:** [Calendar](../objects/calendar.md), [DateTimePicker](../objects/datetimepicker.md)

**Description**


The MinDate property specifies the smallest date that the user may select in a [Calendar](../objects/calendar.md) or [DateTimePicker](../objects/datetimepicker.md) object.


MinDate is an [IDN](../miscellaneous/international-day-number.md) value. Its default value is -109206 which is the minimum date that the Calendar can display.
