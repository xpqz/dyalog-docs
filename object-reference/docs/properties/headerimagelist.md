<h1 class="heading"><span class="name">HeaderImageList</span> <span class="right">Property</span></h1>



**Applies To:** [ListView](../objects/listview.md)

**Description**


The HeaderImageList property specifies the name of or ref  to an  [ImageList](../objects/imagelist.md) object that contains images to be displayed alongside each column heading in a [ListView](../objects/listview.md) object in Report View.
