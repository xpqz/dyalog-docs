<h1 class="heading"><span class="name">AutoPlay</span> <span class="right">Property</span></h1>



**Applies To:** [Animation](../objects/animation.md)

**Description**


Specifies whether or not an AVI clip is played immediately when loaded in an [Animation](../objects/animation.md) object.


AutoPlay is a single number with the value 0 (the default) or 1. If AutoPlay is 1, the AVI clip is automatically played through once from beginning to end when loaded from a file by the [AnimOpen](../methodorevents/animopen.md) method.
