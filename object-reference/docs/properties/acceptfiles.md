<h1 class="heading"><span class="name">AcceptFiles</span> <span class="right">Property</span></h1>

[**Applies To**](../propertyapplies/acceptfiles.md)

**Description**


The AcceptFiles property is Boolean and specifies whether or not an object will accept a file drag/drop operation. Its default value is 0. If set to 1, the object will report a [DropFiles](../methodorevents/dropfiles.md) event when file icons are dropped on it.
