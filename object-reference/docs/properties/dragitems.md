<h1 class="heading"><span class="name">DragItems</span> <span class="right">Property</span></h1>



**Applies To:** [ListView](../objects/listview.md)

**Description**


The DragItems property is Boolean and specifies whether or not the items in a [ListView](../objects/listview.md) object may be drag/dropped by the user. Its default value is 1.
