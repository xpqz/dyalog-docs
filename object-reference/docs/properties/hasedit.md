<h1 class="heading"><span class="name">HasEdit</span> <span class="right">Property</span></h1>



**Applies To:** [BrowseBox](../objects/browsebox.md)

**Description**


Specifies whether or not a [BrowseBox](../objects/browsebox.md) has an edit field.


HasEdit is a single number with the value 0 (the default) or 1. If HasEdit is 1, the user may type in the name of a folder or other resource that is the target of the [BrowseBox](../objects/browsebox.md). If HasEdit is 0, the user must browse to it.
