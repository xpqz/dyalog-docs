<h1 class="heading"><span class="name">PageActive</span> <span class="right">Property</span></h1>



**Applies To:** [PropertySheet](../objects/propertysheet.md)

**Description**


The PageActive property specifies the name of the current [PropertyPage](../objects/propertypage.md) in a [PropertySheet](../objects/propertysheet.md). You may select a particular page by setting this property or by generating a [PageActivate](../methodorevents/pageactivate.md) event.


See also or [PageActiveObject](pageactiveobject.md).
