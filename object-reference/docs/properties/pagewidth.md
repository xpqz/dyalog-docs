<h1 class="heading"><span class="name">PageWidth</span> <span class="right">Property</span></h1>



**Applies To:** [RichEdit](../objects/richedit.md)

**Description**


The PageWidth property specifies the width of the page in a [RichEdit](../objects/richedit.md) object and is the dimension that is used to apply text wrapping and paragraph formatting to the text in the object. PageWidth is a single integer value specified in [Twips](../miscellaneous/twips.md).


The default value of PageWidth is 0 which disables text wrapping. You may find it convenient to set PageWidth to the width of the [RichEdit](../objects/richedit.md) window or to a value that is appropriate for your printer.
